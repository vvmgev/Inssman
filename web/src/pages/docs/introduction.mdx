<h1 className="text-4xl border-b border-neutral-200/20 pb-2 text-neutral-300">Introduction</h1>

With Inssman frontend developers have access to an
<ins className="text-sky-500 hover:text-sky-400">[open-source](https://github.com/vvmgev/Inssman)</ins> platform with essential tools and integrations for writing,
testing, and debugging code more efficiently.
As a result of frontend development, there are usually dependencies on backend developers for API updates,
and reliance on specific environments for comprehensive feature testing. Finally,
there may be occasional dependencies on clients, especially when deploying code on client websites.

<br/>

Inssman now available for Google Chrome and Microsoft Edge browsers.
Frontend developers can reduce their dependency on backend developers, bypass environment complexity,
and control the network layer to inspect, capture and modify https requests & responses.

<br />

Moreover, Inssman is completely free to use, offering an accessible solution without any financial barriers.
Users can create unlimited rules within the platform, providing flexibility and scalability to accommodate diverse project requirements.
