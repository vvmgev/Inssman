<h1 className="text-4xl border-b border-neutral-200/20 pb-2 text-neutral-300">Modify Headers</h1>

<br/>

HTTP Headers consist of key-value pairs that provide extra information in an HTTP request or response.
Developers often aim to easily adjust these headers for the traffic handled by their browsers.

<br/>

<img className="w-full rounded" alt="Modify header" src="/assets/images/rules-image/modify-header.png" />

<br/>

<SourceCondition />

<br/>

<span className="text-neutral-200">**Operator Type:**</span>
This is the section where need to select modification type. You can choose to  <TextBG text="Add"/>, <TextBG text="Remove"/> or <TextBG text="Append"/>.

<br/>

<span className="text-neutral-200">**Request/Response Modification:**</span>
Here need to select <TextBG text="Request"/> or <TextBG text="Response"/> to modify headers.

<br/>

<span className="text-neutral-200">**Request/Response Key:**</span>
This field for header key of <TextBG text="Request"/> or <TextBG text="Response"/>.


<br/>

<span className="text-neutral-200">**Request/Response Value:**</span>
This field for header value of <TextBG text="Request"/> or <TextBG text="Response"/>.

<br/>

It is possible to add a HTTP Header multiple times. The rule will work if one of them matches the source URL.
In each HTTP Header, there is an option to enable or disable it.
